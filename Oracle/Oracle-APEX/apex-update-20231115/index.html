Oracle APEX 23.2 正式发布

发表于 2023-11-15 | 分类于 Oracle ， Oracle APEX | 阅读次数：
本文字数： 2.6k | 阅读时长 ≈ 4 分钟

好消息！！Oracle APEX 23.2 现已提供官方下载 ，并将在全球 Oracle云基础设施 (OCI) 区域推出。

新版本让开发人员能够以低代码的方式轻松构建安全且可扩展的企业云、移动端以及SaaS应用。开发者现在有能力访问各种原生的企业级特性和功能，包括端到端的流程自动化、协作开发和新的UI组件。

以下是 Oracle APEX 23.2 用于企业级应用开发的亮点：

  • 端到端的原生工作流支持

从这个版本开始，APEX拥有了完全集成的工作流解决方案。开发人员能够使用原生内置的直观、可视化工作流设计器实现业务流程和工作流程的自动化，包括系统到系统、人与系统的交互。

  • 协作开发：应用程序工作副本

原生的团队开发功能支持，使开发人员有能力通过创建应用程序的工作副本来修复错误或添加功能，有选择地将其更改并合并回主应用程序，进而解决多个开发人员并行开发同一应用程序过程中发生的冲突问题。

  • 新的UI控件

新的页面控件（包括QR二维码生成器、图片上传和组合框控件）为企业级应用程序提供了无需编码的全新功能。

此版本还包括许多附加功能和错误修复，从而为APEX应用程序带来更高的质量和稳定性。有关Oracle APEX 23.2中的新功能以及许多其他令人兴奋的增强功能的更多信息，请参阅发行说明或访问 新增功能页面。您还可以访问 Oracle APEX Office Hours页面观看APEX 23.2新功能介绍视频。

Oracle APEX由超过70万名开发人员组成的充满活力的社区提供支持，他们正在构建复杂的应用程序来解决众多企业面临的最严峻的挑战。这些开发人员在APEX方面取得的成功和产品发展方向中发挥着重要作用，我们很高兴在APEX 23.2中分享已经实现的来自20多个开发人员请求的想法。APEX开发人员正在构建的系统涵盖了从ERP、内容管理、CRM、物流管理、付款处理、欺诈检测到简单的电子表格替换应用程序等众多领域，与使用“高代码”技术相比，开发人员通过APEX能够以20倍的速度构建具有世界级的企业应用，代码量减少了100倍 ，同时将应用程序开发工作量减少了65%。

低代码应用开发者也在寻找利用人工智能为其组织提供新的尖端功能的方法。据Gartner®称，“到2024年，40%的企业应用程序将嵌入对话式AI，而2020年这一比例还不到5%。”

Oracle APEX使开发者能够通过RESTful API的方式与OCI AI服务进行轻松打通，进而快速构建下一代人工智能驱动的应用系统。这些人工智能服务包括：Oracle Generative AI、AI Speech、AI Vision、Oracle Document Understanding、Oracle Digital Assistant和Oracle Data Science。通过Oracle APEX和OCI AI服务相结合，开发人员可以收益于强大的AI带来的个性化体验和建议、处理自然语言提示、自动执行图像和对象检测、生成文本（电子邮件、营销文案等）。

APEX 可用性和定价

Oracle APEX在OCI（Oracle所有全球云区域）中作为独立、完全托管的服务以按使用量付费模式提供服务。您只需为您消耗的资源付费，无须为每个应用或每个用户付费。OCI的始终免费服务中也包含了相应的免费版本。同时，APEX还免费包含在任何Oracle数据库服务中，包括自治数据库（ADB）。特定功能不收取额外费用 - 所有功能均可供所有客户使用。
