镜像内容

钢哥注：这个镜像是国外 Oracle APEX 大神做的，原始 Github 地址在这里。由于安装介质下载往往耗时耗力，因此钢哥这里仅仅为大家简单打包了一下，并提供国内下载地址，镜像解压后大约会占用 20 GB 的磁盘空间。
该镜像仅用于学习 Oracle APEX 使用，不提供任何技术支持，也不包含任何 License，试用 24 小时后请自行删除。

该镜像的内容包括：

  • Oracle Linux 7.5 64位
  • Oracle Database 18.3 企业版（non-CDB 架构）
  • Oracle APEX 18.1.0
  • Oracle ORDS 18.2
  • Apache Tomcat 8.0.53
  • Java JDK 8u181

确保宿主机已安装好了docker，如果还没有安装的同学可以看这篇文章：《Docker - 通往新世界的大门》
