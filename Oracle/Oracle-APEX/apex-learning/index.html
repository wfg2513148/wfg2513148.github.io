Oracle APEX Foundations 官方免费视频课程

发表于 2022-11-17 | 分类于 Oracle ， Oracle APEX | 阅读次数：
本文字数： 1.2k | 阅读时长 ≈ 2 分钟

为了广大Oracle APEX爱好者更好地学习APEX，Oracle大学（OU）学习社区近期开放了免费的“Oracle APEX Foundations”课程学习路径。在这里您可以学习到如何使用Oracle APEX进行低代码应用程序开发，并了解如何构建具有世界一流特性的、可以在任何地方部署，且可扩展和安全的企业应用程序。

课程内容

在此学习路径中，您将学习到：

  • 创建 APEX 服务实例
  • 配置 APEX Workspace
  • 在 OCI 上使用 APEX 构建云应用程序
  • 使用 SQL Workshop 创建和管理数据库对象
  • 在应用程序中创建和管理报告、表单和其他页面
  • 管理您的应用程序安全性
  • 部署应用程序

学习路径

apex.oracle.com/go/foundations

没有Oracle账号的同学可以在线注册一下，点击页面右上角的Account，并点击Sign up for an Oracle Account，然后根据提示注册即可。

如果您已有Oracle账号，则在上述页面直接点击Sign in to my account进行登录。

登录成功后，首先点击Enroll in this path，注册课程。

课程注册成功后，将页面定位到Your goal: Pass Assessment位置，并点击相应的课程进行学习即可（课程总时长3小时49分钟）。

友情提示：虽然授课语言是英文的，可以通过打开中文字幕来学习，所以英文不好的同学也不用担心。

学习完成后，别忘记点击Assessment: Oracle APEX: Foundation检测下自己的学习效果。通过评估后，即可获得 Oracle APEX Foundations 徽章，可以在社交网络上分享该徽章，以展示您在该技术方面的专业知识。还等什么，赶紧加入到 Oracle APEX 的学习大军中吧！
