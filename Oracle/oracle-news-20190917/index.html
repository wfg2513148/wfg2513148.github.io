Oracle提供始终免费的自治数据库和云基础架构

发表于 2019-09-17 | 分类于 Oracle | 阅读次数：
本文字数： 2k | 阅读时长 ≈ 3 分钟

新的 Always Free 服务使开发人员和学生能够无限时间学习，构建和获得 Oracle 云的实践经验

甲骨文今天宣布推出 Oracle云免费套餐，包括新的免费服务，任何人都可以无限时间尝试世界上第一个自动驾驶数据库和Oracle云基础架构。现在，大型和小型组织，开发人员，学生和教育工作者可以构建，学习和探索Oracle自治数据库和Oracle云基础架构的全部功能，包括计算虚拟机，块和对象存储以及负载均衡器 - 所有必需品开发人员在 Oracle Cloud 上构建完整的应用程序。

虽然其他超大规模云供应商提供了一个为期12个月的关系数据库免费试用版，然后开始收费，但Oracle的 Always Free Autonomous Database 只要使用它就可以免费使用。与竞争对手的免费试用相比，Oracle还提供更多计算和存储作为Always Free服务。用户可以轻松升级以获得更多实例，更大的实例和其他服务。始终免费服务在世界所有地区都可用，任何人都可以使用，包括那些使用通用信用卡定价和新免费套餐帐户的付费帐户。

“我们很高兴能够提供始终免费的Oracle自治数据库和Oracle云基础架构，”Oracle数据库服务器技术执行副总裁 Andrew Mendelsohn 说。“这使下一代开发人员，分析师和数据科学家能够学习最新的数据库和机器学习技术，以便在云上开发强大的数据驱动应用程序和分析。”

新程序使开发人员能够在Oracle云基础架构和自治数据库之上使用任何语言和框架构建应用程序。他们可以快速入门，无需等待IT人员提供和学习人工智能和机器学习等新技术。企业可以使用 Free Tier 进行原型设计，证明新技术，并在将生产工作负载迁移到云之前进行测试。他们可以采用强大的企业基础架构功能，如负载平衡和存储克隆。此外，学生可以学习如何使用最新技术，并为他们的职业生涯做好更充分的准备。

Oracle还提供了一组丰富的免费开发人员工具，因此开发人员可以在Oracle自治数据库上更快地构建和部署新的数据驱动应用程序。开发工具包括 Oracle Application Express（APEX低代码的Web应用程序的开发）、SQL Developer Web的网络与数据库，用户交互的机器学习笔记本，REST接口，方便地访问和数据库数据的发布，以及所有流行的编程语言的驱动程序。在 Oracle开发者云镜像 包括最新的工具，选择开发语言，OCI软件开发工具包（SDK），以及数据库连接器。Oracle云基础架构资源都可以通过控制台运行，REST API和SDK，或使用Terraform自动化。

作为今天新闻的一部分，Oracle还将通过 Oracle Academy，全球慈善教育计划，为教育工作者和学生提供 Oracle Cloud Free Tier。从2020年初开始，Oracle Academy 机构成员将享受轻松注册，教育工作者将能够为他们的学生创建免费帐户。此外，Oracle Academy 正在开发Oracle自治数据库和 Oracle APEX 的学术课程。

Oracle的 Free Tier 程序有两个组件：

  • 始终免费服务，可无限制地访问Oracle云服务；
  • 免费试用，30天内提供300美元的积分，以尝试额外的服务和更大的配置；

新的Always Free计划包括用户在云中构建和测试应用程序所需的基本知识：Oracle自治数据库，计算VM，块卷，对象和归档存储以及负载均衡器。规格包括：

  • 2个自治数据库（自治数据仓库或自治事务处理），每个都有1个OCPU和20 GB存储；
  • 2计算VM，每个VM具有1/8 OCPU和1 GB内存；
  • 2个Block Volumes，总共100 GB，最多5个免费备份；
  • 10 GB对象存储，10 GB存档存储和50,000 /月API请求；
  • 1个负载均衡器，10 Mbps带宽；
  • 10 TB /月的出站数据传输；
  • 5亿摄取数据点和10亿监测服务数据点；
  • 每月100万通知递送选项和每月1000封电子邮件；

Oracle拥有世界上最受欢迎的数据库之一，尤其适用于任务关键型应用程序。Oracle自治数据库以及Oracle的所有云服务和Oracle应用程序在Oracle云基础架构上运行最佳。Oracle自治数据库是业界第一个也是唯一一个自驱动数据库，旨在运行在Oracle的Gen 2基础架构上，几乎每个数据库和工作负载在Oracle云基础架构的性能，网络和安全性方面都能更好地运行。

Oracle自治数据库是第一个使用突破性的机器学习来提供自动驾驶，自我修复和自我保护功能。因此，它可以自动化数据库系统中的密钥管理和安全流程，如修补，调整和升级，同时保持关键基础架构不断运行以获得现代云体验。Oracle自治数据库在Oracle云基础架构上运行，其成本远低于其他选择。
