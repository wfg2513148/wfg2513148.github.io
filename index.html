Oracle Database 18c Express Edition (XE) 正式发布！！

发表于 2018-10-22 | 分类于 Oracle ， Database | 阅读次数：
本文字数： 2.4k | 阅读时长 ≈ 4 分钟

为所有人免费提供Oracle数据库

无论您是开发人员、DBA、数据科学家、教育工作者，还是仅仅对数据库感到好奇，Oracle Database 18c Express Edition（XE）都是您入门的理想方式。它与企业在全球范围内所依赖的功能强大的 Oracle 数据库一样，具有下载简单、使用方便以及全功能的体验。您可以在任何环境中使用Oracle数据库，以及嵌入和重新分发的功能 - 并且，所有这些都完全免费！

该版本都包含什么？

多租户：通过管理 Oracle 多租户容器数据库中的多个可插拔数据库，您可以获得隔离、灵活性以及规模经济；

内存数据库：通过将重要数据保存在 Oracle Database In-Memory 列存储中，您将获得实时分析、商业智能和报表的能力；

分区：通过满足不同业务需求的数据分区，提高数据库的性能、可用性和可管理性；

高级分析：您可以使用数据挖掘，R语言编程以及Oracle Data Miner UI，从数据中获取有价值的见解并甚至预测的能力；

高级安全性：从源头保护您的敏感数据，并构建具有多层安全性的端到端加密应用程序，包括 Oracle 透明数据加密和数据编辑功能；

可分配资源情况：

  • 最大支持 12 GB 的用户数据；
  • 最多可分配 2 GB 的数据库内存；
  • 最多可分配 2个 CPU 线程；
  • 最多支持3个可插拔数据库；

有关更多详细信息，请转至Oracle Database 18c Express Edition 许可信息

为什么使用 XE 数据库？

开发者

您是否正在为您的下一个开发项目寻找合适的数据库？使用免费 XE 数据库，您将从中了解有关 Oracle 数据库更多的信息。

  • 将 Oracle 数据库与您最喜欢的编程语言和开发环境连接在一起，包括 Java，.NET，Python，Node.js，Go，PHP，C/C++ 等；
  • 用世界领先的关系数据库中学习 SQL，或者试验 Oracle 对 JSON 文档、空间和图形数据的天然支持特性；
  • 同时拥有 Oracle 提供的免费开发工具和IDE，包括：SQL Developer，SQLcl 和 SQL Developer Data Modeler；
  • 安装免费的 Oracle REST 数据服务（ORDS）以便启用数据库的 REST 功能；
  • 对于低代码应用程序开发，可以在 ORDS 和 XE 之上运行 Oracle APEX，无需额外成本即可快速构建在移动和桌面浏览器中看起来很漂亮的以数据为中心的 Web 应用程序；

数据库管理员

Oracle 数据库中的测试驱动器改进，使得 DBA 和其他使用免费 XE 的管理员的工作更轻松。使用 XE，任何管理员都可以从使用 Oracle 数据库的许多高级功能中受益。

  • 使用 Oracle 的多租户可插拔数据库，让您可以在一个地方同时管理多个 Oracle 数据库；
  • 使用表分区加速数据库查询；
  • 通过数据压缩从数据库存储中获取更多价值；
  • 使用 Oracle RMAN 备份整个数据库，以及
  • 为了安全性和合规性，使用透明数据加密对静态数据进行加密，设置数据库审计策略以跟踪数据访问，并配置 Database Vault 以防止特权用户进行未经授权的访问；

数据科学家

Oracle 数据库为数据科学家提供了纯粹的分析能力，而 XE 拥有所有这一切，让我们看看这都意味着什么：

  • 使用 Oracle In-Memory 列存储可以显著增加对大型数据集的查询速度；
  • 使用 Advanced Analytics（包括数据挖掘）特性将极大地减少复杂分析，以便简化您的 SQL 语句；
  • 在 Oracle 的免费 Data Miner UI 中以图形方式快速构建您的分析；
  • 使用 Data Pump，SQL Loader，外部表或 SQL Developer，使得您可以将数据快速加载到数据库中；
  • 如果您更喜欢用 R 语言编程，Oracle 数据库也将完全支持；

教育从业者

教师和学生可以自由地使用 XE 进行数据库课程和教学。学生可以将它安装在笔记本电脑上随时随地工作，而不是被束缚在计算机实验室里。

  • Oracle Academy提供课件套装；
  • 没有许可费用；
  • 易于安装；
  • 功能齐全的数据库；

独立软件供应商

如果您需要具有所有功能和最小占用空间的 Oracle 数据库，那么 XE 就是您的最佳选择。

  • 它可以用来嵌入待您的软件中；
  • 随您的软件分发；
  • 在客户现场进行概念验证；
  • 为您的潜在客户提供全面的原型支持；

每一个人

无论是大客户，还是小型用户，Oracle Database XE 都非常适合，比如它适合：

  • 需要立即开始开发但预算有限的初创公司；
  • 非营利组织和其他想要 Oracle 数据库的组织，不需要完全受支持的版本；
  • 需要向客户展示其应用程序的软件开发人员，以及
  • 任何想要私有沙箱进行数据库评估，测试和概念验证的人；
