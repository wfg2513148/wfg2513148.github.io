为了广大Oracle APEX爱好者更好地学习APEX，Oracle大学（OU）学习社区近期开放了免费的“Oracle APEX Foundations”课程学习路径。在这里您可以学习到如何使用Oracle APEX进行低代码应用程序开发，并了解如何构建具有世界一流特性的、可以在任何地方部署，且可扩展和安全的企业应用程序。

课程内容

在此学习路径中，您将学习到：

  • 创建 APEX 服务实例
  • 配置 APEX Workspace
  • 在 OCI 上使用 APEX 构建云应用程序
  • 使用 SQL Workshop 创建和管理数据库对象
  • 在应用程序中创建和管理报告、表单和其他页面
  • 管理您的应用程序安全性
  • 部署应用程序

学习路径

apex.oracle.com/go/foundations

没有Oracle账号的同学可以在线注册一下，点击页面右上角的Account，并点击Sign up for an Oracle Account，然后根据提示注册即可。

如果您已有Oracle账号，则在上述页面直接点击Sign in to my account进行登录。

登录成功后，首先点击Enroll in this path，注册课程。

课程注册成功后，将页面定位到Your goal: Pass Assessment位置，并点击相应的课程进行学习即可（课程总时长3小时49分钟）。

友情提示：虽然授课语言是英文的，可以通过打开中文字幕来学习，所以英文不好的同学也不用担心。

学习完成后，别忘记点击Assessment: Oracle APEX: Foundation检测下自己的学习效果。通过评估后，即可获得 Oracle APEX Foundations 徽章，可以在社交网络上分享该徽章，以展示您在该技术方面的专业知识。还等什么，赶紧加入到 Oracle APEX 的学习大军中吧！

Oracle APEX 22.1 全新发布

发表于 2022-05-04 更新于 2022-05-10 分类于 Oracle ， Oracle APEX 本文字数： 1.3k 阅读时长 ≈ 5 分钟

Oracle APEX 22.1 现已全新发布！

Oracle APEX 是世界上最受欢迎的企业级低代码开发平台，使您能够构建可扩展且安全的企业应用，同时兼具一流的特性，只要有个能联网的浏览器，就可以在任何地方进行开发测试和部署。在2022年Gartner Peer Insights“客户之声”评选活动中，Oracle AEPX作为企业级低代码应用开发平台，第三次获得了客户选择奖！

22.1新版本引入了审批控件和统一任务列表、简化的创建页面向导、JSON或YAML格式的可读应用程序导出、PWA定制化服务和持久身份验证、以及内置数据生成器、开发人员体验增强等更新组件和服务！

以下是Oracle APEX 22.1中的功能亮点，请访问APEX网站了解有关此版本中所有新增功能。

审批和统一任务清单

新的审批组件让您可以在有限的时间里交付需要人工进行审批的解决方案。您可以轻松地在应用程序中定义任务，同时批准者有能力使用统一任务列表对待审批流程进行处理。新的Sample Approvals应用程序拥有您开始所需的一切！Youtube相关视频

简化页面创建

简化的创建页面向导可以让您比以往更轻松地在创建APEX页面，并通过为应用赋予默认值、最小化用户输入、改进页面导航等方面进一步简化向导页面的操作。

JSON或YAML格式的可读应用程序导出

新的应用程序导出功能包含YAML/JSON格式且易于解读的信息。SQLcl和APEXExport(Java)实用程序已更新并可以支持新的JSON和YAML导出格式。

PWA渐进式Web应用程序和增强的用户体验

  • 开发人员现在可以通过在PWA中注入自己的服务代码，进而实现消息推送、在线支付、后台更新等任何事情。Youtube视频

  • 您现在可以通过新的持久身份验证功能让您的客户更快地恢复行动。登录一次，保持登录状态。Youtube视频

  • 通用主题样式更新了全新的图像区域模板、对区域图像的支持并改进了页面渲染性能。

全新的和更新的组件

  • 行级搜索功能现在已得到了显著增强，可以为您提供更自然和直观的搜索体验。Youtube视频

  • 新的页面控件排序功能可以使开发人员更轻松地调整区域控件位置。Youtube视频

  • 分面搜索功能也得到了更新，支持客户端过滤的搜索词高亮显示、分面的新操作菜单、额外的分面级控件，进一步提升了用户体验。

  • 同时，无论是HTML还是Markdown，在富文本的创建和显示方面均得到了增强。

开发者体验

  • 使用APEX中全新的数据生成器，将可以轻松定义数据生成蓝图并使用随机生成的样本数据测试您的表格。定义好蓝图后，您可以选择将示例数据导出生成CSV或JSON文件，或将数据直接插入到现有表中。

  • 应用程序图标现在可以在创建应用程序时就进行上传，并可以从开发人员工具栏或共享组件界面进行更新。Youtube视频

  • 现在，您无需离开APEX即可安装入门应用程序和示例应用程序。在几秒钟内，您可以选择您想要的应用程序并从更新的应用程序库中安装它。

  • 现在，您可以在支持REST的SQL之上创建REST源以在远程数据库上执行，并从APEX组件中抽象出SQL查询。Youtube视频

  • 您现在可以将列表从一个应用程序订阅到另一个应用程序，从而使您拥有跨多个应用程序共享的单个列表定义。我们同时还简化了创建和复制列表向导以及编辑列表页面。

  • JavaScript和CSS文件（URL语法）已通过新功能得到增强，包括新的URL指令、注释及改进的编辑体验。Youtube视频

  • Session会话覆盖特性可以让开发人员在会话中快速更改APEX变量值，例如：文件路径、附加文件和全球化属性，从而为开发者提供极大的便利。Youtube视频
