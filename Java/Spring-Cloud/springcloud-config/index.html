Github webhooks

测试Github的webhooks时，网上说是要配置到/monitor这个地址上，实测发现会报错，这里重写了refresh服务的接口，采用get方式接收webhooks调用，然后再转成post方式去请求真实的配置刷新接口：http://kwang.natapp1.cc/refresh （natapp是个内网穿透工具）
